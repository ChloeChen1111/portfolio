Chloe-AI (Live Link)

Chloe-AI is a AI-SaaS project, which stands for Artificial Intelligence Software as a Service. It's a dynamic web application that I've built with impressive features.

I've integrated OpenAI's APIs to bring cutting-edge AI capabilities right to the fingertips of users. They can engage in interactive conversations, generate images and code, and even create music and videos with simple text inputs using Replicate APIs. For monetization, I've streamlined payment processing and subscriptions using Stripe. My user authentication is rock-solid with Clerk, and I provide real-time customer support through Crisp.

What's more, Chloe-AI is designed to be fully compatible with mobile screens and tablet screens, ensuring a seamless user experience on a variety of devices. It's a versatile platform designed to cater to developers, creatives, businesses, and individuals, making it an innovative solution for a wide range of needs.

Citi Wiki (Live Link)

Citi Wiki is a small search engine that specifically helps users search for the cities in the United State along with a wikipedia link attached in order to explore the city with one simple click.

It was developed with JavaScript, HTML5, and CSS. It also utilized a public API to fetch data of all cities in the US and transformed the data into a search engine web app.

Rainbow Pen (Live Link)

Rainbow Pen is an online drawing board that allows users to enjoy drawing or writing with this rainbow pen with adjustable line width, saturation and lightness.

The magic behind the rainbow pen was mainly because of the HTMl Canvas and Colors HSL. HSL provides three properties: hue, saturation, and lightness, in order to present the color-changing pen.
